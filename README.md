LinkedIn Cringe Translator

Turn everyday sentences into overly polished, dramatic LinkedIn-style posts.

This project mimics a Google Translate interface but instead of translating languages, it transforms plain English into exaggerated, reflective, and sometimes absurd professional content.

What it does

You input something simple like:

I fell from the stairs

And get something like:

Today reminded me that progress isn’t always linear.

In the middle of an ordinary moment, I experienced an unexpected setback — one that forced me to pause, reflect, and reset.

Growth often comes when we least expect it.

Features

🔄 English → LinkedIn-style text transformation

🎭 Multiple writing modes (cringe, CEO, humble, thought leader)

💀 “Make it worse” mode for maximum exaggeration

🧩 Clean, Google Translate-inspired interface

📋 Copy output instantly

Tech Stack

🌐 Frontend: HTML, CSS, JavaScript

⚙️ Backend: Flask (Python)

🤖 AI: Hugging Face API

🧠 Core logic: Prompt-based text transformation
